Appendix B Installing

Aspell requires gcc 2.95 (or better) as the C++ compiler. Other C++ compilers should work with some effort. Other C++ compilers for mostly POSIX compliant (Unix, Linux, BeOS, Cygwin) systems should work without any major problems provided that the compile can handle all of the advanced C++ features Aspell uses. C++ compilers for non-Unix systems might work but it will take some work. Aspell at very least requires a Unix-like environment (sh, grep, sed, tr, …), and Perl in order to build. Aspell also uses a few POSIX functions when necessary.

The latest version can always be found at GNU Aspell’s home page at http://aspell.net.
